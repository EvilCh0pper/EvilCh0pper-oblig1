## Installation and Running instructions ##

1. Open a terminal and go to a directory where you would like to clone this project unto.

2. Run the following line to clone:  
`git clone https://github.com/EvilCh0pper/EvilCh0pper-oblig1`

3. After cloning the project, change directory to the project folder

4. Run `python -m venv hello_health_venv`

5. Staying in the same directory, activate the virtual environment by running this:  
`hello_health_venv/Scripts/activate`

6. Run `python hello_health.py`

7. Deactivate the virtual environment by running `deactivate`
